# An exercise app

This is a very basic [Progressive Web App] that hosts a UI to facilitate
exercise repetitions and rest periods.

## Design goals

I mean, obviously, I made this for myself.

- Privacy focused; your data never leaves your device.
- Configurable with YAML for a semi-human-readable experience. The pain of
  using YAML—which is real—somewhat addressed through an online validator as
  one edits in the UI.


[Progressive Web App]: https://en.wikipedia.org/wiki/Progressive_web_app
